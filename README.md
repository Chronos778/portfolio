# Maithil Patil - 3D Interactive Portfolio

A premium, immersive 3D developer portfolio built with Next.js, React Three Fiber, and GSAP.

## About

Personal portfolio website showcasing my projects, skills, and experience as a Computer Science student specializing in Data Science and AI/ML. The site breaks away from traditional scrolling websites by offering a fully interactive WebGL 3D environment where visitors can smoothly navigate through my professional journey in a 3-dimensional space.

## Tech Stack

- **Next.js 15+** - React framework with App Router
- **React 19** - Latest React with improved performance
- **TypeScript** - Type-safe development
- **Three.js & React Three Fiber** - 3D rendering and scene management
- **@react-three/drei** - Useful helpers for R3F
- **GSAP** - Professional-grade smooth animations and camera transitions
- **Tailwind CSS** - Utility-first styling
- **Vercel** - Seamless deployment and hosting

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Clone the repository:

```bash
git clone https://github.com/Chronos778/Portfolio.git
cd Portfolio
```

1. Install dependencies:

```bash
npm install
```

1. Run the development server:

```bash
npm run dev
```

1. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

```bash
npm run build
npm start
```
